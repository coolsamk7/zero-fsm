# ⚡ Zero-FSM

A **zero-boilerplate, type-safe finite state machine (FSM) and workflow engine** for TypeScript.

## ✨ Features
- 🎯 Simple, declarative API
- 🛡️ Type-safe states & events
- ⚡ Async transitions
- 🔗 Workflow composition
- ✅ Built with TDD

## 🚀 Getting Started

```bash
npm install zero-fsm
